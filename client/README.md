# React-JS--Crud-Mysql
## Installation

### Client

  1. `$ cd client`
  2. `$ npm install` or `$ yarn install`
  3. `$ npm start` or `$ yarn start`
  
### Server

  1. Change your mySQL database data `server/index.js`
  2. `$ cd server`
  3. `$ npm install` or `$ yarn`
  4. `$ npm rum devStart`
  
## Functionalities

  + Create cia in the database
  + Read cia in the database
  + Update cia in the database
  + Delete cia in the database
  + Click card to edit and delete
